# Gatsby + MDX + MDX Deck + Netlify で手軽にスライドを公開する

---

## Gatsby とは

[公式サイト](https://www.gatsbyjs.org/docs/)

- Static Site Generator (以下 SSG) のツール群
- [Jamstack](https://jamstack.org/)を支えるソフトウェアの一つ

---

## MDX とは

[公式サイト](https://mdxjs.com/)

- Markdown のシンタックスに JSX を追加したドキュメント

```mdx
# Hello, _world_!

Below is an example of JSX embedded in Markdown. <br /> **Try and change
the background color!**

<div style={{ padding: '20px', backgroundColor: 'tomato' }}>
  <h3>This is JSX</h3>
</div>
```

---

## MDX Deck とは

[公式サイト](https://github.com/jxnblk/mdx-deck)

- MDX でプレゼンテーションを作成できるソフトウェア

---

## Netlify とは

[公式サイト](https://docs.netlify.com/)

- ホスティングサービスの一つ
- 静的なサイトを簡単にビルド、デプロイして公開することが出来る
- Git レポジトリからインポート出来るのが便利

---

## 準備

```bash
yarn global add gatsby
gatsby new slides
cd slides
yarn add gatsby-plugin-mdx gatsby-theme-mdx-deck gatsby-plugin-typescript
yarn add -D typescript
```

今回は Gatsby を使用するので、ライブラリを直接追加しない。
