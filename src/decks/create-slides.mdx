# Gatsby + MDX + MDX Deck + Netlify で手軽にスライドを公開する

---

## Gatsby とは

[公式サイト](https://www.gatsbyjs.org/docs/)

- Static Site Generator (以下 SSG) のツール群
- [Jamstack](https://jamstack.org/)を支えるソフトウェアの一つ

---

## MDX とは

[公式サイト](https://mdxjs.com/)

- Markdown のシンタックスに JSX を追加したドキュメント

---

## MDX Deck とは

[公式サイト](https://github.com/jxnblk/mdx-deck)

- MDX でプレゼンテーションを作成できるソフトウェア

---

## Netlify とは

[公式サイト](https://docs.netlify.com/)

- ホスティングサービスの一つ
- 静的なサイトを簡単にビルド、デプロイして公開することが出来る
- Git レポジトリからインポート出来るのが便利
